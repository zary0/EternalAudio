プライバシーポリシー

最終更新日: 2025年9月24日

1. はじめに

Eternal Audio（以下「本アプリ」）は、お客様のプライバシーを尊重し、保護することを重要視しています。本プライバシーポリシーは、本アプリがどのような情報を収集し、どのように使用するかを説明します。

2. 当社のプライバシーに関する基本方針

本アプリは、お客様の音声データや生成コンテンツを一切収集しません。

すべての音声処理はお客様のデバイス上でローカルに実行されます。

3. 収集する情報

3.1 自動的に収集される情報

利用統計情報（匿名）

  • アプリのバージョン番号
  • OSの種類とバージョン
  • アプリの起動回数（個人を特定しない形式）
  • 機能の使用頻度（個人を特定しない形式）

ライセンス検証情報

  • Microsoft StoreのライセンスID（購入確認のため）
  • デバイスID（ハッシュ化された形式、1台制限の確認のため）

3.2 収集しない情報

以下の情報は一切収集しません：

  • 音声録音データ
  • 音声認識されたテキスト
  • 生成されたコンテンツ
  • 個人名、メールアドレス、電話番号
  • 位置情報
  • 連絡先情報
  • カメラやマイクの録音内容（ローカル処理のみ）

4. 情報の使用目的

収集した匿名情報は以下の目的でのみ使用します：

  • アプリの品質向上とバグ修正
  • 新機能の開発優先度の決定
  • ライセンスの正当性確認
  • 利用統計の把握（完全に匿名化された形式）

5. データの保存場所

5.1 ローカルデータ

  • 録音ファイル: お客様のデバイスのみ（AppData/Local/EternalAudio/recordings）
  • 設定情報: お客様のデバイスのみ（AppData/Local/EternalAudio/settings）
  • これらのデータは当社のサーバーには送信されません

5.2 サーバー側データ

  • 匿名の利用統計のみ（Cloudflare Workers KV）
  • 個人を特定できる情報は保存しません

6. 第三者との情報共有

6.1 共有しない情報

お客様の個人情報、音声データ、生成コンテンツは第三者と共有しません。

6.2 外部サービスの利用

  • OpenAI API: お客様が独自のAPIキーを使用する場合、OpenAIのプライバシーポリシーが適用されます
  • Microsoft Store: ライセンス検証のため、Microsoft Storeと通信しますが、個人情報は送信しません
  • GitHub Releases: アップデート確認のみ（ダウンロード統計に含まれる可能性があります）

7. データセキュリティ

7.1 ローカルセキュリティ

  • すべての音声処理はローカルで実行
  • 録音データは暗号化されずに保存されます（お客様のOSのファイルシステムセキュリティに依存）
  • OpenAI APIキーはWindowsの資格情報マネージャーに安全に保存

7.2 通信セキュリティ

  • アップデート確認とライセンス検証はHTTPS通信を使用
  • 最小限の情報のみを送信

8. お子様のプライバシー

本アプリは13歳未満のお子様を対象としていません。13歳未満のお子様から意図的に個人情報を収集することはありません。

9. プライバシーポリシーの変更

本ポリシーを変更する場合は、アプリ内での通知またはGitHubリポジトリでお知らせします。重要な変更の場合は、変更前に通知期間を設けます。

10. データの削除

10.1 ローカルデータの削除

  • アプリをアンインストールすることで、すべてのローカルデータが削除されます
  • 録音ファイルは手動で削除することも可能です

10.2 サーバー側データ

  • 匿名統計データのため、特定の個人に関連付けられたデータは存在しません

11. お客様の権利

お客様には以下の権利があります：

  • 本アプリの使用を中止する権利
  • ローカルデータを削除する権利
  • プライバシーに関する質問をする権利

12. Cookie の使用

本アプリはCookieを使用しません。

13. Do Not Track

本アプリはDo Not Track設定を尊重します（そもそもトラッキングを行いません）。

14. カリフォルニア州消費者プライバシー法（CCPA）

本アプリは個人情報を販売しません。

15. EU一般データ保護規則（GDPR）

EU在住のお客様の場合：

  • データ管理者: 該当なし（個人データを収集しないため）
  • 法的根拠: 該当なし（個人データを処理しないため）
  • データ保護責任者: 不要（個人データを扱わないため）

16. お問い合わせ

プライバシーに関するご質問は以下までお問い合わせください：

  • GitHub Issues: https://github.com/RyotaOzawa0/eternal-audio/issues
  • Microsoft Store レビュー機能

17. 同意

本アプリを使用することにより、本プライバシーポリシーに同意したものとみなされます。
